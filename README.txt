My SBA project is a floral theme website created with html and css. It includes a homepage, shop page, and contact page. I use a flexbox container on the three images of the website for them to have consistent positioning in the home page.

Technical Specifications include:

Three pages

Dropdown menu

Complementary color palette

Two forms(newsletter signup,contact form)

Two css animations using transition(box shadow and transform)

Include text, images, and a GIF

Inline, internal, external css styling.

The use of flex box.
